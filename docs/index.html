Tab to Window / Popup

Dragging is a drag, move your tabs to a new window in an instant!

Moving Tabs

Tabs can be moved to a new window with either a key command or by clicking the extension button.

You can move multiple tabs by selecting them first (by holding shift or command/control).

Moving between windows

Tabs can also be moved between windows with the Tab to Next Window command.

Window Type

New windows can also be a Popup window. This can be assigned to a separate key command, or selected in the extension button menu.

Window Arrangement

The window arrangement can be customised in the options page.

Resizing the original window is optional.

You can also choose whether to focus on the Original or New window after splitting.

Multiple Displays

Multiple displays are supported! The windows will stay on their current display.

A tab can be moved to a new window on the next display with the Tab to Next Display command.

Key Commands

The key commands can be customised by following the link on the options page.

The defaults are:

Mac

  • AltX - Window
  • AltC - Popup
  • AltZ - Next Window
  • AltD - Next Display

Windows

  • AltX - Window
  • AltC - Popup
  • AltZ - Next Window
  • AltShiftd - Next Display

Linux

  • AltShiftX - Tab to Window
  • AltShiftC - Tab to Popup
  • AltShiftZ - Tab to Next Window
  • AltShiftd - Tab to Next Display
